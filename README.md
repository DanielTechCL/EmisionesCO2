# EmisionesCO2
This Python program calculates the estimated CO₂ emissions for a Hyundai Santa Fe 2018 based on the distance traveled in kilometers. Users input a number over 0, and the program outputs the total CO₂ in kilograms. Type "stop" to exit. Designed for beginner practice with loops, conditionals, and basic calculations.

Este programa de Python calcula las emisiones estimadas de CO₂ de un Hyundai Santa Fe 2018 según la distancia recorrida en kilómetros. Al introducir un número mayor que 0, el programa muestra el CO₂ total en kilogramos. Para salir, pulse "stop". Diseñado para principiantes con bucles, condicionales y cálculos básicos.
