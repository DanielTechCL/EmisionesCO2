# EmisionesCO2
# Calculadora de Emisiones de CO₂

Este proyecto es un programa en Python que calcula las emisiones estimadas de CO₂ de un Hyundai Santa Fe 2018 según los kilómetros recorridos. Está diseñado para principiantes que quieren practicar conceptos básicos de Python como bucles, condicionales y manejo de entradas de usuario.

## Cómo funciona
- El usuario ingresa el número de kilómetros recorridos.
- El programa calcula las emisiones de CO₂ en kilogramos usando un factor de 0.215 kg de CO₂ por kilómetro.
- Si se ingresa 0 o un número negativo, el programa solicita un número válido.
- Escriba `stop` para salir del programa.

## Características
- Bucle continuo para realizar múltiples cálculos.
- Validación básica de entradas.
- Ideal para aprender y practicar programación básica en Python.

## Tecnologías
- Python 3

## Notas
Este proyecto es educativo y está pensado como práctica para principiantes. Ayuda a entender cómo recibir datos del usuario, realizar cálculos simples y usar bucles y condicionales en Python.



# CO₂ Emissions Calculator

This project is a Python program that calculates the estimated CO₂ emissions of a 2018 Hyundai Santa Fe based on the distance traveled. It is designed for beginners who want to practice basic Python concepts such as loops, conditionals, and handling user input.

## How it works
- The user enters the number of kilometers traveled.

- The program calculates the CO₂ emissions in kilograms using a factor of 0.215 kg of CO₂ per kilometer.

- If 0 or a negative number is entered, the program prompts for a valid number.

- Type `stop` to exit the program.

## Features
- Continuous loop for performing multiple calculations.

- Basic input validation.

- Ideal for learning and practicing basic Python programming.

## Technologies
- Python 3

## Notes
This project is educational and intended as practice for beginners. It helps to understand how to receive user data, perform simple calculations, and use loops and conditionals in Python.
